# payment_gateway.php (Mock Philippine Payment System)

- This is a demo payment gateway for your car detailing booking system.
- Supports GCash, Maya, Bank Transfer, and Over-the-Counter (7-Eleven, Cebuana, MLhuillier).
- All payment methods are simulated for academic use.
- No real money is charged.
- For production, replace with PayMongo or real payment API.
